Goal

Consider the set of genes with sig. difference in translation efficiency between human and chimpanzee. We then take the set of genes that are buffered at the translation level and count the number of genes that are classifed as buffered at the protein level. We apply the same steps for non-buffered genes at the translation level.

These analyses are for investigating whether divergence at the translation level is likely to be carried over to the protein level.

Problem

For the definition of buffering an non-buffering genes, we consider the following two approaches:

  • Interaction approach: Genes are classifed as buffered at the translation or at the protein level if there is a significant interaction between Ribo or protein and RNA inter-species differences and if the effect size of the inter-species difference at the RNA level is greater than the effect size of the inter-species differene at the Ribo or protein level.

  • Size approach: Genes are classifed as buffered at the translation or at the protein level if the effect size of the inter-species difference at the RNA level is greater than the effect size of the inter-species difference at the Ribo or protein level.

Definitions

Take the comparison between RNA and Ribo for example, we describe the definition of buffered and non-buffered genes as follows:

  • Interaction approach:

    1. Buffered at Ribo level: Sig. RNA-Ribo interaction AND ( abs(RNA divergence) > abs(Ribo divergence) )

    2. Non-buffered at Ribo leve: Sig. RNA-Ribo interaction AND ( abs(RNA divergence) < abs(Ribo divergence) )

  • Size approach:

    1. Buffered at Ribo level: ( abs(RNA divergence) > abs(Ribo divergence) )

    2. Non-buffered at Ribo leve: ( abs(RNA divergence) < abs(Ribo divergence) )

fc.mat <- lapply( seq_along( c("human","chimp") ), function(i) {
  ii <- eSet.temp$species == c("human","chimp")[i]
  eSet.tt <- eSet.temp[, ii]
  emat <- lapply( seq_along( c("ribo","rna","protein") ), function(j) {
    jj <- eSet.tt$seqData == c("ribo","rna","protein")[j]
    rowMeans( exprs(eSet.tt[, jj] ), na.rm=TRUE )
  })
  emat <- do.call(cbind, emat)
  colnames(emat) <- c("ribo","rna","protein")
  return( data.frame(emat) )
})
names(fc.mat) <- c("human","chimp")

We look further into the genes that were buffered at the Ribo level and had no significant inter-species difference between RNA and Protein.

Question: We see in the scatter plot, some genes have a much larger effect size at the RNA level compared to the protein level. Yet, these genes did not show significant difference between inter-species differences at the RNA level and and the protein level.

A possible explanation: Maybe these genes have a larger variance at the RNA level compared to the protein level. Because statistial significance is a function of effect size and variance, perhaps the size of the variances of these genes canceled out the large effect size of these genes.
