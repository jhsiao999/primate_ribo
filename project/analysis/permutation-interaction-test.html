Goal

Permutation-based significance testing for the interaction models. We observed in our differential divergence analysis comparing species difference between phenotypes (RNA vs. Ribo, RNA vs. Protein, and Ribo vs. Protein) an odd p-value distribution (http://jhsiao999.github.io/primate_ribo/project/analysis/compare-ribo-rna-quantified-in-sequencing.html). Therefore, we decided to compute permutation-based p-values for all the interaction tests. The goal of the permutation is to construct an empirical null distribution for the interaction effect. Three permutation schemes are proposed as follows:

  1. Shuffle across species and technology: For each gene and each inter-species comparison of RNA, Ribo or protein levels, we shuffle the sample labels across species and also technology type for 1000 times. The empirical p-value is computed as the proportion of permutation-based test statistic that is greater than the observed test statistic. Likelihood ratio test statistic (chi-squared) is used to test the interaction effect.

  2. Shuffle within speciees across technology: For each gene and each inter-species comparison of RNA, Ribo or protein levels, we shuffle the sample labels with each species and across techology for 1000 times. The empirical p-value is computed as the proportion of permutation-based test statistic that is greater than the observed test statistic. Likelihood ratio test statistic (chi-squared) is used to test the interaction effect.

  3. Shuffle within technology across species: For each gene and each inter-species comparison of RNA, Ribo or protein levels, we shuffle the sample labels with each technology type and across species for 1000 times. The empirical p-value is computed as the proportion of permutation-based test statistic that is greater than the observed test statistic. Likelihood ratio test statistic (chi-squared) is used to test the interaction effect.

We used the permute-interact-per-gene function in the Humanzee package for generating the permuted data sets and interaction results are recorded in permute-interact-per-gene.r. Each permutation set of results of 3,188 genes was produced on midway using 1 clustering and 10 nodes, taking 3 and a half hours to completion.

Helper function for computing empirical p-value

R source 
get_perm_pvals <- function(perm_data, obs_data) {
    n_simu <- length(perm_data)
    n_genes <- dim(obs_data)[1]
    perm_pvalues <- sapply(1: n_genes, function(per_gene) {
                            n_larger <- sapply(perm_data, function(per_permute) {
                                            per_permute[per_gene, "LRatio"] > obs_data[per_gene, "LRatio"]
                                        })
                            n_larger <- sum(n_larger, na.rm = TRUE)
                            if (n_larger > 0) {
                                pval <- n_larger/n_simu
                            } 
                            if (n_larger ==0) {
                                pval <- 1/n_simu
                            }
                            return(pval)
                          })
  return(perm_pvalues)
}

For each gene i, the null distribution of its LR statistic is the distribution of the LR statistics across the 1,000 permutations.
